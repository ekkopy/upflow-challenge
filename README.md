## Uso:
<br>

01 - Clone:
```bash
$ git clone https://github.com/thiiagoms/upflow-challenge
```

02 - Vá para o diretório do projeto:
```bash
$ cd upflow-challenge
upflow-challenge $
```

03 - Abra o arquivo `index.html` com o seu navegador favorito, exemplo:
```bash
$ firefox index.html
```
